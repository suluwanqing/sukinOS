# 文件管理

## 基本信息

| 字段 | 值 |
|---|---|
| 资源ID | `sys-fileSystem-demo` |
| 应用类型 | `system` |
| 寄生模式 | 是 |
| 暴露状态 | 否 |

## 功能说明

系统级应用，提供虚拟盘、本地盘、云端文件浏览与管理。支持目录导航、文件创建/重命名/删除、右键菜单、列表/网格视图切换。

## 注册配置

- **快捷方式**：支持 (`hasShortcut: true`)
- **持久化**：不保存状态 (`saveState: false`)

## 依赖

- `AppSDK.Components.FileSystem`
