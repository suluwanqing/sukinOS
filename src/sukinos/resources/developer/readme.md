# 开发者中心

## 基本信息

| 字段 | 值 |
|---|---|
| 资源ID | `sys-developer-demo` |
| 应用类型 | `system` |
| 寄生模式 | 是 |
| 暴露状态 | 否 |

## 功能说明

系统级应用，提供开发者工具与调试面板。通过 SDK Components 注入 `<Developer />` 组件渲染。

## 注册配置

- **快捷方式**：支持 (`hasShortcut: true`)
- **持久化**：不保存状态 (`saveState: false`)

## 依赖

- `AppSDK.Components.Developer`
