# Notification 组件使用指南

Notification 是一个悬浮通知卡片组件，采用标题层、内容层、底边操作层三层结构设计，支持自定义内容及 5 种屏幕定位。

## 基础用法

提供标题和内容即可渲染完整卡片：

```jsx
import Notification from '@/component/notification/layout'

function App() {
  return (
    <Notification
      title='系统提示'
      content='这是一条基础的通知内容描述信息。'
      onClose={() => console.log('关闭通知')}
    />
  )
}
```
## Props API

| 参数      | 说明                                              | 类型                                                                         | 默认值      |
| --------- | ------------------------------------------------- | ---------------------------------------------------------------------------- | ----------- |
| position  | 通知弹出的位置（5 种屏幕定位）                    | 'top-right' \| 'top-left' \| 'bottom-right' \| 'bottom-left' \| 'top-center' | 'top-right' |
| title     | 标题层文字内容                                    | React.ReactNode                                                              | -           |
| icon      | 标题层左侧的图标（React 节点）                    | React.ReactNode                                                              | -           |
| content   | 内容层文字（当存在 children 时优先显示 children） | React.ReactNode                                                              | -           |
| children  | 内容层自定义 Body 区域（React 节点）              | React.ReactNode                                                              | -           |
| footer    | 底边操作层内容（React 节点）                      | React.ReactNode                                                              | -           |
| showClose | 是否显示右上角关闭按钮                            | boolean                                                                      | true        |
| onClose   | 点击关闭按钮时的回调函数                          | (e: Event) => void                                                           | -           |
| onClick   | 点击整个通知卡片时的回调函数                      | (e: Event) => void                                                           | -           |
| className | 额外的 CSS 类名                                   | string                                                                       | ''          |
| style     | 额外的内联样式（会透传到根节点）                  | React.CSSProperties                                                          | -           |
