# InfoListn 组件使用指南

`InfoListn` 是用于常规的综合平台分列的资源列表渲染。

## 基础用法


```jsx
import Input from '@/component/common/list/infoList/layout'
import { useState } from 'react'

function MyForm() {
  const [value, setValue] = useState('')
  const infoListMenuData=[
    {
      nav: { text: '推荐', hot: true },
      content: [
          {
          title: '仓库资源',
          items: [
                  { text: 'H5 静态实例', type: 'outer', path: 'https://gitee.com/suluwan/animation_source' },
          ]
          },
      ]
    }
    ]

  return (
    <InfoList
      menuData={infoListMenuData}
      onItemClick={handleInfoListItemClick}
      style={{ ...positionInfolist }}
      onMouseLeave={handleSomeOtherMouseLeave}
      />
  )
}
```
## 属性 Props

| 属性名         | 类型       | 默认值     | 必填 | 描述                 |
| -------------- | ---------- | ---------- | ---- | -------------------- |
| `menuData`     | `Array`    | `[]`       | 否   | 菜单数据数组         |
| `onItemClick`  | `Function` | `() => {}` | 否   | 内容项点击回调函数   |
| `onMouseLeave` | `Function` | `() => {}` | 否   | 鼠标离开组件回调函数 |
| `style`        | `Object`   | `{}`       | 否   | 自定义样式对象       |

## 数据结构

### menuData 结构

```javascript
menuData = [
  {
    nav: {
      text: String, // 导航项显示文本
      active: Boolean, // 是否默认激活
      hot: Boolean // 是否显示 HOT 标签
    },
    content: [
      // 右侧内容分组
      {
        title: String, // 分组标题
        items: [
          // 分组项列表
          {
            text: String, // 项显示文本
            sub: Boolean, // 是否为子项
            isNew: Boolean // 是否显示 NEW 标签
          }
        ],
        subGroup: {
          // 子分组（可选）
          title: String, // 子分组标题
          items: [
            // 子分组项列表
            {
              text: String,
              sub: Boolean
            }
          ]
        }
      }
    ]
  }
]
```
