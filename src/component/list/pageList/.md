# PageList 组件使用指南

`PageList` 是一个功能强大的数据列表组件，内置了分页、选择、嵌套树形结构、列显隐控制、自动/手动数据管理、深度对象访问以及多种单元格类型渲染（图片、视频、链接等）功能。

## 基础用法

```jsx
import PageList from '@/component/list/pagelist/layout'

function MyTable() {
const columns = [
  { label: '项目编号', prop: 'id', width: 80 },
  { label: '名称', prop: 'name' },
  { label: '详情', prop: 'meta.desc', deep: true } // 支持深度访问
]

const data = [
  { id: '001', name: '极简主义框架', meta: { desc: '专注于黑白美学' } },
  { id: '002', name: '响应式系统', meta: { desc: '适配全端设备' } }
  ]

return (
<PageList
      theme="default"
      columns={columns}
      data={data}
      total={2}
      selectable
    />
)
}

```
## Props
###  基本配置
| 参数     | 类型   | 默认值 | 是否必填 | 说明                               |
| -------- | ------ | ------ | -------- | ---------------------------------- |
| theme | String | default | 否 | 风格 ，primary，success
，danger
，dark
，flat
，minimal
|
| columns  | Array  | []     | 是       | 列配置数组（详见下方 Column 配置） |
| data     | Array  | []     | 否       | 数据源数组（手动模式下使用）       |
| total    | Number | 0      | 否       | 数据总条数（手动模式下使用）       |
| current  | Number | 1      | 否       | 当前页码（手动模式下使用）         |
| pageSize | Number | 10     | 否       | 每页条数（手动模式下使用）         |
| hasDivider | Boolean | false | 否 | column 是否需要分割线 |
| defaultColumnWidth | Number | 150 | 否 | 默认下,最小column宽度 |



### 数据模式配置

| 参数            | 类型     | 默认值 | 是否必填 | 说明                                     |
| --------------- | -------- | ------ | -------- | ---------------------------------------- |
| isAuto          | Boolean  | false  | 否       | 是否开启自动数据管理模式                 |
| request         | Function | null   | 否       | 自动模式下的数据请求函数，需返回 Promise |
| requestDataPath | String   | null   | 否       | 自动模式下数据路径，如 "data.list"       |

### 数据行配置
| 参数               | 类型    | 默认值     | 是否必填 | 说明                         |
| ------------------ | ------- | ---------- | -------- | ---------------------------- |
| rowKey             | String  | 'id'       | 否       | 数据行的唯一标识字段名       |
| childrenColumnName | String  | 'children' | 否       | 树形结构子节点的字段名       |
| selectable         | Boolean | false      | 否       | 是否开启行选择功能           |
| textMaxLength      | Number  | 0          | 否       | 文本最大显示长度，0 为不限制 |

### 操作列配置

| 参数        | 类型   | 默认值 | 是否必填 | 说明                               |
| ----------- | ------ | ------ | -------- | ---------------------------------- |
| actions     | Array  | []     | 否       | 操作列配置（详见下方 Action 配置） |
| actionTitle | String | '操作' | 否       | 操作列标题                         |
| actionWidth | Number | 150    | 否       | 操作列宽度                         |

### 回调函数

| 参数              | 类型     | 默认值   | 是否必填 | 说明                             |
| ----------------- | -------- | -------- | -------- | -------------------------------- |
| onPageChange      | Function | (page) => {} | 否       | 页码改变回调 (page)              |
| onPageSizeChange  | Function | (size) => {} | 否       | 每页条数改变回调 (size)          |
| onSelectionChange | Function | (rowKey) => {} | 否       | 选中项改变回调 (rowKey) |

## columns
```jsx
{
  label: String, // 表头显示文字
  prop: String, // 数据字段名
  width: Number, // 列宽（像素）
  deep: Boolean, // 是否支持深度路径访问（如 'info.age'）
  type: String, // 单元格类型：'text'(默认) | 'image' | 'video' | 'link'  暂不支持单独自定义,可以使用完全渲染修改
  placeholder: String,// 数据为空时的占位符
  render: Function // 自定义渲染函数 (record) => ReactNode  //他将完全覆盖样式,注意了这个返回的是数据row因为可能会有多条共同判断
}
//
{
    label: '状态',
    prop: 'status',
    width: 120,  //默认填充flex:'1 1 0%'
    render: (record) => (
    <span style={{
              color: reCord.status === '正常' ? 'gray' : 'red',
              background: reCord.status === '正常' ? 'red' : 'white',
              padding: '2px 8px',
              borderRadius: '4px',
              border: `1px solid ${reCord.status === '正常' ? 'gray' : 'red'}`
            }}>
    {reCord.status} //这里需要对应这个prop
    </span>
    )
}
```
