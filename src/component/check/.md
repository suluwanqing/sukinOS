# Check 开关组件文档

## 概述

这是一个高度可定制的开关切换组件（Toggle/Switch），支持多种主题颜色、尺寸、交互模式（标准/圆点）以及视觉风格（简约/暗黑）。它通过 BEM 规范进行样式管理，确保在不同 UI 场景下的表现一致性。

## 功能特性

- **多模式交互**：支持标准滑块模式和 Dot（圆点）模式
- **丰富的尺寸**：提供大、中、小三种预设尺寸
- **视觉风格**：支持 round（全圆角）、plain（简约边框）和 dark（暗黑/硬核）主题
- **状态管理**：完备的 disabled（禁用）状态支持
- **无障碍支持**：内置 aria-checked 和 role="switch" 属性

## 基础用法

最简单的用法是直接引入并使用 Check 组件。


```jsx
import React, { useState } from 'react';
import Check from '@/component/check/layout';
  function MyComponent() {
    const [checked, setChecked] = useState(false);
    return (
    <Check
      checked={checked}
      onChange={setChecked}
      round // 推荐开启全圆角
    />
  );
}

```

组件接收以下属性进行配置：
### check 开关组件 API
## Props API 参考

| 属性名    | 说明                                | 类型                       | 可选值                                                               | 默认值    |
| --------- | ----------------------------------- | -------------------------- | -------------------------------------------------------------------- | --------- |
| checked   | 开关当前的开启/关闭状态             | boolean                    | -                                                                    | false     |
| onChange  | 状态变化时的回调函数                | (checked: boolean) => void | -                                                                    | -         |
| type      | 主题类型，决定激活时的颜色          | string                     | 'primary', 'success', 'warning', 'danger', 'dark', 'info', 'default' | 'default' |
| size      | 开关物理尺寸                        | string                     | 'large', 'medium', 'small'                                           | 'medium'  |
| dot       | 是否开启圆点交互模式           | boolean                    | -                                                                    | false     |
| round     | 是否开启全圆角（药丸形状）          | boolean                    | -                                                                    | false     |
| plain     | 是否为简约风格（透明背景+带色边框） | boolean                    | -                                                                    | false     |
| disabled  | 是否禁用交互                        | boolean                    | -                                                                    | false     |
| className | 额外的自定义 CSS 类名               | string                     | -                                                                    | ''        |
| style     | 自定义内联样式                      | React.CSSProperties        | -                                                                    | -         |
