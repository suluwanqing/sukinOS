# CheckGroup 组件

## 概述

CheckGroup 是一个复选 / 单选 开关组组件（switch-like / toggle-like），用于一组 Check 控件的组合管理，支持单选、多选、最小/最大选择限制，以及受控 / 非受控两种使用方式。

主要特性

- 支持 mode="single"（单选）与 mode="multiple"（多选）
- 支持 min / max 限制（最小/最大选择数量）
- 支持受控（传入 value + onChange）和非受控（不传 value，内部管理）模式
- 支持 option 为简单字符串或完整对象形式，单项可自定义 disabled、type、size、plain、round 等
- 可通过 className / style 自定义外层样式，支持横向布局（is-horizontal）

## 基础用法

1. 单选模式

```jsx
import CheckGroup from '@/component/checkGroup/layout'

// 父组件（受控示例）
const [selected, setSelected] = useState('a')

<CheckGroup
  mode='single'
  value={[selected]}
  onChange={values => setSelected(values[0])}
  options={[
    { label: '选项 A', value: 'a' },
    { label: '选项 B', value: 'b' },
    { label: '选项 C', value: 'c' }
  ]}
/>
```

2. 必须选择 2-4 个

```jsx
const [values, setValues] = useState(['1', '2'])

;<CheckGroup
  mode='multiple'
  min={2}
  max={4}
  value={values}
  onChange={setValues}
  options={[
    { label: '选项 1', value: '1' },
    { label: '选项 2', value: '2' },
    { label: '选项 3', value: '3' },
    { label: '选项 4', value: '4' },
    { label: '选项 5', value: '5' }
  ]}
/>
```

3. 最多选择 2 个

```jsx
<CheckGroup
  max={2}
  value={selectedValues}
  onChange={setSelectedValues}
  options={[
    { label: '选项 1', value: '1' },
    { label: '选项 2', value: '2' },
    { label: '选项 3', value: '3' }
  ]}
/>
```

4. 至少选择 1 个

```jsx
<CheckGroup
  min={1}
  value={selectedValues}
  onChange={setSelectedValues}
  options={[
    { label: '选项 1', value: '1' },
    { label: '选项 2', value: '2' },
    { label: '选项 3', value: '3' }
  ]}
/>
```

5. 非受控（内部管理选中项）

```jsx
<CheckGroup options={['a', 'b', 'c']} />
```

## Props

组件接收以下属性进行配置：

| 属性名    | 说明                                                                               | 类型                           | 默认值     |
| --------- | ---------------------------------------------------------------------------------- | ------------------------------ | ---------- |
| value     | 受控选中值数组（当传入时组件为受控）                                               | any[]                          | undefined  |
| onChange  | 选中值变化回调：onChange(newValues: any[])                                         | (newValues: any[]) => void     | undefined  |
| options   | 选项列表，支持字符串/数字或对象数组                                                | Array<string\|number\|Option>  | []         |
| type      | Check 的风格类型，会被传给子项                                                     | string                         | 'default'  |
| size      | Check 的尺寸，会被传给子项                                                         | 'small' \| 'medium' \| 'large' | 'medium'   |
| plain     | Check 是否为朴素风格，会被传给子项                                                 | boolean                        | false      |
| round     | Check 是否为圆角样式，会被传给子项                                                 | boolean                        | false      |
| disabled  | 整个组是否禁用（true 则所有项都不可交互）                                          | boolean                        | false      |
| mode      | 模式：'multiple'（多选）或 'single'（单选）                                        | 'multiple' \| 'single'         | 'multiple' |
| min       | 多选模式下的最少选择数；在 single 模式下可用以禁止取消（min===1 表示必须选中一个） | number                         | undefined  |
| max       | 多选模式下的最多选择数                                                             | number                         | undefined  |
| className | 外层容器自定义 class                                                               | string                         | ''         |
| style     | 外层容器内联样式                                                                   | React.CSSProperties            | {}         |
