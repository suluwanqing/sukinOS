# button组件

## 概述
简单的利用bem处理的常规按钮组件


## 基础用法

最简单的用法是直接引入并使用 `Button` 组件。

```jsx
import Button from '@/component/button/layout'

function MyComponent() {
  return <Button>默认按钮</Button>
}
```
## Props
组件接收以下属性进行配置：

| 参数           | 说明                                                                | 类型                                                                                    | 默认值      |
| :------------- | :------------------------------------------------------------------ | :-------------------------------------------------------------------------------------- | :---------- |
| `children`     | 按钮的显示内容，可以是文本或任何 React 节点                         | `React.ReactNode`                                                                       | `-`         |
| `type`         | 按钮类型，决定按钮的主题颜色                                        | `string` <br/> `'primary'`, `'success'`, `'warning'`, `'danger'`, `'info'`, `'default'` | `'default'` | `'dark'`|
| `size`         | 按钮尺寸                                                            | `string` <br/> `'large'`, `'medium'`, `'small'`                                         | `'medium'`  |
| `plain`        | 是否为朴素按钮（背景透明，带边框）                                  | `boolean`                                                                               | `false`     |
| `round`        | 是否为圆形按钮                                                      | `boolean`                                                                               | `false`     |
| `disabled`     | 是否禁用按钮                                                        | `boolean`                                                                               | `false`     |
| `loading`      | 是否显示加载中状态（会覆盖 `icon` 并禁用按钮）                      | `boolean`                                                                               | `false`     |
| `icon`         | 自定义图标，可以传入一个 React 组件（如 SVG 组件）                  | `React.ReactNode`                                                                       | `-`         |
| `iconPosition` | 图标显示位置                                                        | `string` <br/> `'left'`, `'right'`                                                      | `'left'`    |
| `className`    | 额外的自定义 CSS 类名，会合并到组件根元素上                         | `string`                                                                                | `''`        |
| `...restProps` | 接受所有原生 `<button>` 元素的属性，如 `onClick`, `type`, `name` 等 | `React.ButtonHTMLAttributes`                                                            | `-`         |
