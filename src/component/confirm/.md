# React 全局确认对话框 (Confirm) 组件文档

## 概述

基于 React 的命令式 API 确认对话框组件，支持全局调用和现代化动画效果。

## 功能特性

-  全局调用：任意 JS 模块均可触发
-  灵活内容：支持文本/自定义组件
-  输入支持：可选输入框功能
-  现代化设计：暗色主题 + 流畅动画
-  回调处理：`onConfirm`/`onCancel`
-  交互友好：点击遮罩可取消
-  Portal 渲染：避免样式干扰
-  BEM 命名：样式隔离

## 安装配置


在任何 JavaScript 文件中导入并使用 `confirm` 对象：

#### 基础用法：

```Jsx
import { confirm } from '@/component/confirm/layout';
function handleClick() {
  confirm.show({
      title: '确认删除',
      content: '您确定要删除这条记录吗？删除后将无法恢复。',
      onConfirm: () => {
      console.log('用户点击了确定');
      // 执行删除逻辑
      },
      onCancel: () => {
            console.log('用户点击了取消');
      }
  });
  }
```
#### 带输入框的用法：

```Jsx
import { confirm } from '@/component/confirm/layout';
function handleRename() {
  confirm.show({
        title: '重命名文件',
        content: '请输入新的文件名：',
        showInput: true,
        inputPlaceholder: '例如：report.docx',
        inputDefaultValue: '旧文件名.txt',
        onConfirm: (fileName) => {
        if (fileName) {
            console.log(`新的文件名是: ${fileName}`);
        // 执行重命名逻辑
        } else {
           console.log('用户没有输入文件名。');
        }
        },
  });
  }
```

#### 自定义输入框
```Jsx
const CustomContent = () => (
    <div>
    <h3>自定义区域</h3>
    <p>使用任何React组件作为内容</p>
    </div>
);

confirm.show({
  title: "自定义对话框",
  children: <CustomContent />
});

```


## API 参考

### `confirm.show(config)`

显示确认对话框。
### confirm.show(config)

| 参数              | 类型      | 默认值                 | 说明           |
| ----------------- | --------- | ---------------------- | -------------- |
| title             | string    | "提示"                 | 对话框标题     |
| content           | string    | "确定要执行此操作吗？" | 正文内容       |
| children          | ReactNode | null                   | 自定义组件内容 |
| showInput         | boolean   | false                  | 是否显示输入框 |
| inputPlaceholder  | string    | ""                     | 输入框占位文本 |
| inputDefaultValue | string    | ""                     | 输入框默认值   |
| onConfirm         | Function  | () => {}               | 确认回调       |
| onCancel          | Function  | () => {}               | 取消回调       |
|     showCancel| Boolean |false | 是否单列即不显示取消 |

### confirm.hide()

手动关闭对话框

## 实现原理

1. **命令式调用**：通过 `useImperativeHandle` 暴露 API
2. **状态管理**：内部使用 React 状态管理
3. **动画系统**：CSS 类控制 + `onAnimationEnd` 事件
4. **Portal 渲染**：`createPortal` 到 body 元素

## 最佳实践

### `confirm.hide()`

手动隐藏确认对话框（通常不需要手动调用，内部会自动处理）。

## 实现原理

- **命令式调用**：通过 `React.createRef` + `useImperativeHandle` 暴露方法给外部调用。
- **状态管理**：使用 `useState` 管理对话框的显示/隐藏状态和配置。
- **Portal 渲染**：使用 `createPortal` 将对话框渲染到 `document.body`，避免受父组件样式影响。
