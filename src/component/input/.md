# Input 组件使用指南

`Input` 是一个功能丰富的输入框组件，支持前后缀、一键清空、密码显隐等常用功能。它基于 BEM 规范，并通过 CSS 变量实现主题化。

## 基础用法

最简单的用法是作为一个标准的输入框。它支持受控和非受控两种模式。

```jsx
import Input from '@/component/input/layout'
import { useState } from 'react'

function MyForm() {
  const [value, setValue] = useState('')

  return (
    <Input
      placeholder='请输入内容'
      value={value}
      onChange={e => setValue(e.target.value)}
    />
  )
}
```

---

## Props API

| 参数           | 说明                                                       | 类型                             | 默认值     |
| :------------- | :--------------------------------------------------------- | :------------------------------- | :--------- |
| `value`        | 输入框内容（受控）                                         | `string`                         | `-`        |
| `defaultValue` | 输入框默认内容（非受控）                                   | `string`                         | `-`        |
| `onChange`     | 内容变化时的回调                                           | `function`                       | `-`        |
| `size`         | 输入框尺寸                                                 | `'large'`, `'medium'`, `'small'` | `'medium'` |
| `isRound`      | 是否为圆边                                               | `Boolean`                         | `false` |
| `disabled`     | 是否禁用                                                   | `boolean`                        | `false`    |
| `clearable`    | 是否可一键清空                                             | `boolean`                        | `false`    |
| `showPassword` | 是否显示密码切换按钮（`type='password'`时生效）            | `boolean`                        | `false`    |
| `prefixIcon`   | 前置图标或内容                                             | `React.ReactNode`                | `-`        |
| `suffixIcon`   | 后置图标或内容（优先级高于`clearable`和`showPassword`）    | `React.ReactNode`                | `-`        |
| `className`    | 额外的 CSS 类名                                            | `string`                         | `''`       |
| `maxLength` |   最长长度限制                                                | `number`                       |   `-` |
| `minLength` | 最短长度限制                                                 | `number`                        |     `-` |
| `...restProps` | 原生 `<input>` 属性，如 `placeholder`, `type`, `maxLength` | `object`                         | `-`        |
---

### 带图标的输入框

使用 `prefixIcon` 和 `suffixIcon` 添加前后缀内容。

```jsx
// 假设 <UserIcon /> 和 <SearchIcon /> 是 SVG 图标组件
import { UserIcon, SearchIcon } from './icons';

<Input placeholder="请输入用户名" prefixIcon={<UserIcon />} />
<Input placeholder="搜索" suffixIcon={<SearchIcon />} />
```
### 注意事项
- 我们对input事件没有做这个对value的处理而是直接返回了这个e事件
