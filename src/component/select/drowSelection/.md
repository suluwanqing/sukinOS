# Select 组件使用指南

`Select` 是一个下拉选择器组件，支持自定义尺寸、下拉方向、选项配置及外部点击关闭等功能。

## 基础用法

```jsx
import Select from '@/component/select/drowSelection/layout'
import { useState } from 'react'

function MyComponent() {
  const [value, setValue] = useState('1')

  const options = [
    { label: '选项一', value: '1' },
    { label: '选项二', value: '2' },
    { label: '选项三', value: '3' }
  ]

  return (
    <Select
      value={value}
      options={options}
      onChange={setValue}
      placeholder='请选择内容'
    />
  )
}
```

# 组件 API 文档

## Props
showTitle

| 参数          | 类型             | 默认值    | 必填 | 说明                                                |
| ------------- | ---------------- | --------- | ---- | --------------------------------------------------- |
| value         | String \| Number | -         | 是   | 当前选中的值                                        |
| options       | Array            | []        | 否   | 选项数据数组，配置见下方 Option 配置                |
| onChange      | Function         | () => {}  | 否   | 选中值改变时的回调 `(value) => void`                |
| size          | String           | 'default' | 否   | 组件尺寸，可选值：`'large'`, `'default'`, `'small'` |
| direction     | String           | 'bottom'  | 否   | 下拉菜单展开方向，可选值：`'top'`, `'bottom'`       |
| placeholder   | String           | '请选择'  | 否   | 未选中时的占位文本                                  |
| boxStyle      | Object           | {}        | 否   | 触发器容器（输入框）的自定义内联样式                |
| dropdownStyle | Object           | {}        | 否   | 下拉菜单容器的自定义内联样式                        |
| optionStyle   | Object           | {}        | 否   | 单个选项的自定义内联样式                            |
| showTitle | Boolean | false | 否 | 是否悬浮显示title |


## Option 配置

选项数据数组中的每个对象应包含以下属性：

| 参数     | 类型             | 说明                  |
| -------- | ---------------- | --------------------- |
| label    | String           | 选项显示的文本        |
| value    | String \| Number | 选项对应的值          |
| disabled | Boolean          | (可选) 是否禁用该选项 |
