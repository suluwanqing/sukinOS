# Select 组件使用指南

`Select` 是一个下拉选择器组件，支持自定义尺寸、下拉方向、选项配置、外部点击关闭、搜索过滤及异步加载。

## 基础用法

```jsx
import Select from '@/component/select/drowSelection/layout'
import { useState } from 'react'

function MyComponent() {
  const [value, setValue] = useState('1')

  const options = [
    { label: '选项一', value: '1' },
    { label: '选项二', value: '2' },
    { label: '选项三', value: '3' }
  ]

  return (
    <Select
      value={value}
      options={options}
      onChange={setValue}
      placeholder='请选择内容'
    />
  )
}
```

## 可搜索模式（本地过滤）

```jsx
<Select
  value={value}
  options={options}
  onChange={setValue}
  searchable
  placeholder='搜索选项...'
/>
```

## 异步搜索模式（远程加载）

```jsx
<Select
  value={value}
  onChange={setValue}
  searchable
  optionsAsync={async ({ page, pageSize, searchQuery }) => {
    const res = await fetch(`/api/users?page=${page}&size=${pageSize}&q=${searchQuery}`)
    return { items: res.data, total: res.total }
  }}
  placeholder='搜索用户...'
/>
```

`optionsAsync` 接收对象 `{ page, pageSize, searchQuery }`，应返回 `{ items: [{ label, value }], total: number }`。支持分页，列表底部会出现"加载更多"按钮。

## Props

| 参数 | 类型 | 默认值 | 必填 | 说明 |
|------|------|--------|------|------|
| value | String \| Number | - | 是 | 当前选中的值 |
| options | Array | `[]` | 否 | 选项数据数组，见下方 Option 配置 |
| onChange | Function | `() => {}` | 否 | 选中值改变时的回调 `(value) => void` |
| size | String | `'default'` | 否 | 组件尺寸：`'large'`、`'default'`、`'small'` |
| direction | String | `'bottom'` | 否 | 下拉展开方向：`'top'`、`'bottom'` |
| placeholder | String | `'请选择'` | 否 | 未选中时的占位文本 |
| boxStyle | Object | `{}` | 否 | 触发器容器的自定义内联样式 |
| dropdownStyle | Object | `{}` | 否 | 下拉菜单容器的自定义内联样式 |
| optionStyle | Object | `{}` | 否 | 单个选项的自定义内联样式 |
| searchable | Boolean | `false` | 否 | 是否启用搜索输入框。options 模式本地过滤，optionsAsync 模式远程加载 |
| optionsAsync | Function | `null` | 否 | 异步加载选项函数，见上方异步搜索模式说明。启用后替换 options |
| showTitle | Boolean | `false` | 否 | 是否悬浮显示 title 提示 |

## Option 配置

| 参数 | 类型 | 说明 |
|------|------|------|
| label | String | 选项显示的文本 |
| value | String \| Number | 选项对应的值 |
| disabled | Boolean | (可选) 是否禁用该选项 |

## 行为说明

- **options 模式**：选项通过 `options` 属性传入，`searchable` 开启时在本地按 `label` 过滤
- **optionsAsync 模式**：选项通过 `optionsAsync` 异步函数加载，`searchable` 必须同时开启。支持输入防抖（300ms）、分页滚动加载
- **外部点击关闭**：点击组件外部区域自动关闭下拉菜单并重置搜索状态
- **自动聚焦**：打开下拉时搜索输入框自动获得焦点
