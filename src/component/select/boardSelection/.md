# BoardSelection 组件

这是一个用于“选择打开方式”的通用面板组件。它具有以下特性：

1.  **居中显示**：使用 Flex 布局固定在屏幕中央。
2.  **自动滚动**：当选项过多超过屏幕高度（`max-height: 80vh`）时，内部列表会自动出现滚动条。
3.  **遮罩层交互**：点击背景半透明区域自动关闭，点击面板内部不会关闭。
4.  **BEM 命名**：适配你的 `createNamespace` 工具。

---

## 1. 组件代码 (BoardSelection.jsx)

```jsx
const MyComponent = () => {
// 控制显示隐藏的状态
const [showBoard, setShowBoard] = useState(false);

// 定义选项数据
const appList = [
{ id: 'vscode', label: 'VS Code', icon: '💻' },
{ id: 'chrome', label: 'Google Chrome', icon: '🌏' },
{ id: 'notepad', label: '记事本', icon: '📝' },
];

// 处理选中逻辑
const handleSelect = (item) => {
console.log('用户选择了:', item.label);
setShowBoard(false); // 选中后通常需要关闭面板
};

return (
<div>
<button onClick={() => setShowBoard(true)}>
打开方
</button>

      {/* 渲染组件 */}
      <BoardSelection
        visible={showBoard}
        title="选择打开文件的应用"
        options={appList}
        onSelect={handleSelect}
        onClose={() => setShowBoard(false)}
      />
    </div>

);
};

export default MyComponent;
```
### Props

#### 组件参数

| 参数名   | 类型     | 必填 | 默认值   | 说明                                                |
| -------- | -------- | ---- | -------- | --------------------------------------------------- |
| visible  | Boolean  | 是   | false    | 控制弹窗是否显示                                    |
| options  | Array    | 是   | []       | 选项数据数组，具体结构见下文 `options` 数据结构说明 |
| onSelect | Function | 是   | -        | 点击某一项时的回调函数，参数为被点击的选项对象      |
| onClose  | Function | 是   | -        | 点击遮罩层或取消按钮时的回调函数                    |
| title    | String   | 否   | "请选择" | 弹窗顶部的标题文本                                  |
|noneInfo|   String|  否 |   '无可用选择'|无数据时展示|
---

#### options 数据结构

`options` 数组中的每一项应包含以下字段：

| 字段名 | 类型             | 说明                                       |
| ------ | ---------------- | ------------------------------------------ |
| id     | String / Number  | 选项的唯一标识符（用于 React 的 key 属性） |
| label  | String           | 选项显示的文本名称                         |
| icon   | ReactNode (可选) | 显示在选项左侧的图标或自定义组件           |
