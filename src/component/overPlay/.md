# 包裹组件 遮盖层

## 概述

这是一个用于遮盖的,采用flex方案不使用,fixed方案
```jsx
import OverPlay from '@/component/common/overPlay' //根据实际修改
import useState from "react"
function Item(){
  return (
    <div>需要被包裹的</div>
  )
}
function Container(){
  const [isOpen,setIsOpen]=useState(false)

  return (
        <div>
         {isOpen && ( <OverPlay closeFunc={()=>setIsOpen(!isOpen)}>
                          <Item/>
                    </OverPlay>
                )
          }
        </div>
  )
}
```


## API 参考
| Prop        | 类型      | 必需 | 默认值     | 描述                                                |
| :---------- | :-------- | :--- | :--------- | :-------------------------------------------------- |
| `children`  | ReactNode | 是   | -          | 需要操作的子组件。         |
| `style` | 对象 | 否 | - | 需要自定义遮盖层的样式。 |
| `closeFunc` | 函数 | 是 | - | 父组件需要控制开关的函数。 |
| `isFixed` | Boolean | 否 | - | 组件是否需要脱离文档流。 |
| `styleC` | 对象 | 否 | - | 直接包裹容器的样式 |
|`index`|Number|否|1000|控制z-index|
|`onlyDom`|Boolean|false|控制是否默认是覆盖dom|
