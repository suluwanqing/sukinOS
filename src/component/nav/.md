# Nav 组件使用指南

Nav 是一个高度可定制的导航菜单组件，支持多级级联、水平/垂直布局、侧边栏折叠及多种色彩主题。基于 BEM 规范开发，纯 CSS 驱动动画交互。

## 基础用法

最简单的垂直导航菜单用法，支持通过 `icon` 节点渲染图标：

```jsx
import Nav from '@/component/nav/layout'
import { useState } from 'react'

const navItems = [
  { id: 'home', label: '首页', icon: <span>🏠</span> },
  { id: 'dashboard', label: '仪表盘', icon: <span>📊</span> },
  {
    id: 'settings',
    label: '系统设置',
    icon: <span>⚙️</span>,
    children: [
      { id: 'profile', label: '个人资料' },
      { id: 'security', label: '安全设置' }
    ]
  }
]

function App() {
  const [current, setCurrent] = useState('home')

  return (
    <Nav items={navItems} activeId={current} onChange={id => setCurrent(id)} />
  )
}
```
### navItem
```js
type NavItem = {
  id: string | number; // 唯一标识符
  label: string; // 菜单显示文本
  icon?: React.ReactNode; // 菜单图标（React 节点）
  children?: NavItem[]; // 子菜单，支持无限级联
}

```

### props
| 参数       | 说明                        | 类型                   |          默认值                     |
| items      | 导航菜单数据配置项数组                                | Array<{ id, label, icon?, children? }>                          | []         |
| activeId   | 当前激活的菜单项 ID（受控）                           | string \| number                                                | -          |
| onChange   | 点击无子菜单项时触发的回调，返回该项 ID               | (id: string \| number) => void                                  | -          |
| mode       | 导航模式（垂直侧边栏 / 水平顶栏）                     | 'vertical' \| 'horizontal'                                      | 'vertical' |
| theme      | 颜色主题                                              | 'light' \| 'dark' \| 'primary' \| 'cyan' \| 'purple' \| 'green' | 'light'    |
| collapsed  | 是否处于折叠状态（仅 mode="vertical" 生效）           | boolean                                                         | false      |
| onCollapse | 触发折叠/展开的回调（传入此函数才会显示底部折叠按钮） | (collapsed: boolean) => void                                    | -          |
| className  | 额外的 CSS 类名                                       | string                                                          | ''         |
| style      | 额外的内联样式                                        | React.CSSProperties                                             | -          |
