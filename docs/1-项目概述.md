# SukinOS 项目概述

## 一、项目定位

SukinOS 是一个基于 Web 技术的**云端桌面操作系统**，运行在浏览器中，提供类似传统操作系统的桌面环境、应用生态和管理后台。用户可以通过浏览器访问一个完整的桌面界面，在其中运行各类 Web 应用（称为"SukinOS APP"），并可上传、安装、管理自己的应用。

**核心理念**：将操作系统体验搬上云端，实现跨平台的桌面级 Web 应用平台。

## 二、核心功能

### 2.1 桌面环境
- 完整的窗口管理器（拖拽、缩放、最小化、关闭）
- 进程内核（进程生命周期管理、消息通信、沙箱隔离）
- 任务栏与开始菜单
- 桌面快捷方式
- 虚拟文件系统（VFS）

### 2.2 应用生态
- **APP商店**：浏览、搜索、安装、更新应用
- **应用管理**：上传自己的 APP（10MB 限制）、版本管理、审核流程
- **权限注册池**：第三方 APP 可接入权限系统，实现精细化的用户授权
- **在线开发**：内置开发者中心，支持在线 IDE 开发
- **本地开发**：支持与本地开发环境的同步调试（SDK 模式）

### 2.3 内置应用
| 应用 ID | 名称 | 描述 |
|---|---|---|
| sys-store | APP商店 | 浏览和安装应用 |
| sys-start | 开始 | 开始菜单 |
| sys-setting | 设置 | 系统设置 |
| sys-developer | 开发者中心 | 在线开发 IDE |
| sys-local-dev | 本地开发 | APP 本地同步调试 |
| sys-fileSystem | 文件管理 | 文件浏览器 |
| sys-notebook | 记事本 | 文本笔记 |
| sys-drawBoard | 画板 | 绘图与思维导图 |
| sys-sheet | 表格 | 电子表格 |
| sys-systemManage | 系统管理 | 后台管理控制台 |

### 2.4 系统管理后台
- **管理概览**：系统核心指标面板
- **成员中心**：用户搜索、分页展示、编辑资料、分配角色
- **审计分析**：全平台用户行为分析（活跃时段、在线趋势）
- **系统配置**：动态配置管理、预设模板、日志过滤规则、系统 APP 可见性
- **更新日志**：发布编辑和管理系统版本更新日志
- **系统日志**：请求日志审计（多维度过滤、数据趋势分析、CSV 导出）
- **应用管理**：查看、审核、下架、删除第三方应用
- **权限管理**：
  - 角色管理（创建、编辑、删除、引用检查）
  - 路由权限（每个 API 路由的可访问角色/用户配置）
  - 菜单权限（侧边栏导航项可见性控制）
  - APP 权限注册池
  - 注册权管理（放权给非 root 用户操作注册池）
  - 中间件白名单配置

### 2.5 认证与安全
- 双 Token JWT 认证（Access Token + Refresh Token）
- 密码/验证码双模式登录
- 静默 Token 刷新
- 多层级权限控制（中间件、路由、装饰器）
- 请求日志审计（含脱敏、采样）
- 操作人轨迹追溯

## 三、技术栈

### 3.1 后端
| 组件 | 技术 | 说明 |
|---|---|---|
| Web 框架 | FastAPI | 高性能异步 Python Web 框架 |
| 数据库 | MySQL 8.0+ | 关系型数据库 |
| ORM | SQLAlchemy 2.0 | 异步兼容的 ORM 框架 |
| 缓存 | Redis | 验证码缓存、Session 管理 |
| 任务队列 | Celery | 异步任务（验证码发送等） |
| 认证 | JWT (PyJWT) | 双 Token 认证机制 |
| 密码哈希 | Passlib (bcrypt) | 安全的密码存储 |
| 静态文件 | Starlette StaticFiles | 用户头像、APP 文件托管 |

### 3.2 前端
| 组件 | 技术 | 说明 |
|---|---|---|
| 框架 | React 18 | 前端 UI 框架 |
| 语言 | JSX (JavaScript) | 前端开发语言 |
| 构建工具 | Vite | 前端构建与 dev server |
| 状态管理 | Redux Toolkit | 集中式状态管理 |
| 持久化 | redux-persist | 状态持久化到 localStorage |
| 路由 | React Router v6 | 前端路由 |
| 网络请求 | Axios | HTTP 客户端 |
| CSS 方案 | CSS Modules | 样式隔离 |

### 3.3 部署
| 组件 | 技术 |
|---|---|
| 反向代理 | Nginx |
| 进程管理 | systemd |
| 构建产物 | dist/ (静态文件) |

## 四、项目结构

```
B:\reluf/
├── myfastapi_/              # 后端 Python 项目
│   ├── Api/                 # API 路由层
│   │   ├── Common/          # 公共模块（Auth、权限、验证码、文件处理）
│   │   ├── Location/        # 系统管理 API（用户管理、配置、日志、角色等）
│   │   ├── SukinOS/         # SukinOS APP 相关 API（上传、更新、列表）
│   │   ├── SocketApi/       # WebSocket API（在线时长统计）
│   │   └── User/            # 用户 API（登录注册、密码修改）
│   ├── Middleware/           # 中间件（路由权限、静态鉴权、请求日志）
│   ├── Model/               # 数据模型
│   │   ├── DbModel/         # ORM 模型（user、sukinos、system）
│   │   ├── RequestModel/    # 请求参数 Pydantic 模型
│   │   └── ResponseModel/   # 响应体 Pydantic 模型
│   ├── Func/                # 业务逻辑层
│   │   ├── system/          # 系统服务（配置、权限、日志、APP 服务）
│   │   ├── date/            # 日期工具
│   │   └── judge/           # 邮件发送
│   ├── Config/              # 配置（静态配置、系统配置键、CORS 等）
│   ├── CeleryWork/          # Celery 异步任务
│   ├── MyRedis/             # Redis 客户端封装
│   └── resource/            # 用户资源文件（头像等）
├── src/                     # 前端 React 项目
│   ├── apis/                # API 请求层（auth、sukinos、system）
│   ├── apps/                # 第三方/示例应用
│   ├── component/           # 通用组件
│   ├── hooks/               # 通用 Hooks
│   ├── index/               # 首页/登陆页
│   ├── middleware/           # 前端中间件（路由权限、水印等）

├── utils/                   # 全局工具（Service Worker 等）

```
