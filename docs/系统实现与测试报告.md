### 1.2 项目架构概述

SukinOS 是一个运行在浏览器端的**桌面操作系统 Web 应用**，采用前后端分离架构：

- **前端 (b:/sukinOs)**：React 19 + Redux Toolkit，模拟桌面操作系统核心体验——窗口化多任务、应用动态加载、虚拟文件系统、进程间通信及安全沙箱隔离。数据完全存储在用户浏览器本地（IndexedDB）。
- **后端 (b:/myfastapi_)**：FastAPI + SQLAlchemy + Celery + Redis，提供用户认证管理、SukinOS 应用商店、请求日志审计、WebSocket 实时通信、SSE 本地开发热重载等服务。

##### (2) JWT 双 Token 认证系统 (Api/Common/Auth.py)

系统实现了完整的 **Access Token + Refresh Token** 双 Token 机制：

```python:19:55:b:/myfastapi_/Api/Common/Auth.py
def create_access_token(data: dict, remember: bool = True):
    """创建短期 Access Token (用于业务请求)"""
    to_encode = data.copy()
    expires_delta = timedelta(seconds=ACCESS_TOKEN_EXPIRE_MINUTES)
    expire = datetime.now(timezone.utc) + expires_delta
    to_encode.update({
        "exp": int(expire.timestamp()),
        "type": "access",
        "remember": remember
    })
    encoded_jwt = jwt.encode(to_encode, SECRET_KEY, algorithm=ALGORITHM)
    return encoded_jwt, int(expires_delta.total_seconds())

def create_refresh_token(data: dict, remember: bool = True):
    """创建长期 Refresh Token (仅用于刷新 Access Token)"""
    to_encode = data.copy()
    expires_delta = timedelta(seconds=LONG_REFRESH_TOKEN_EXPIRE)
    expire = datetime.now(timezone.utc) + expires_delta
    to_encode.update({
        "exp": int(expire.timestamp()),
        "type": "refresh",
        "remember": remember
    })
    encoded_jwt = jwt.encode(to_encode, SECRET_KEY, algorithm=ALGORITHM)
    return encoded_jwt, int(expires_delta.total_seconds())
```

**认证特性**：
- Access Token：30分钟有效期，用于业务请求
- Refresh Token：7天有效期，用于静默刷新
- 支持"记住我"模式（Cookie max_age 控制）
- 密码使用 bcrypt 哈希存储
- 三级权限模型：角色模式 (role)、位运算权限 (power)、超级管理员 (superuser)

`verify_auth()` 工厂函数实现了**四阶段认证流程**：
1. **第一阶段**：校验 Access Token
2. **第二阶段**：Access Token 无效时，自动用 Refresh Token 静默刷新（生成新 Token 对并注入 Response Cookie）
3. **第三阶段**：基础用户状态校验（is_active 检查）
4. **第四阶段**：详细权限校验（角色/位运算）

```python:7:34:b:/myfastapi_/Api/Re_Rs.py
def to_camel(string: str) -> str:
    """下划线转驼峰"""
    parts = string.split('_')
    return parts[0] + ''.join(word.capitalize() for word in parts[1:])

def convert_to_camel_case(data: Any) -> Any:
    """递归将所有键名转为驼峰"""
    if isinstance(data, list):
        return [convert_to_camel_case(i) for i in data]
    if isinstance(data, dict):
```

```javascript:5:48:b:/sukinOs/src/sukinos/utils/file/fileKernel.js
class fileKernel {
  constructor() {
    this.driver = new fileIndexDb()
    this.inodeMap = new Map()   // 内存镜像: id -> Inode
    this.treeMap = new Map()    // 目录树索引: parentId -> Set<childId>
    this.events = new EventTarget()
    this.ready = false
  }
  async boot() {
    await this.driver.mount()
    const inodes = await this.driver.loadAllInodes()
    this.inodeMap.clear()
    this.treeMap.clear()
    if (inodes.length === 0) {
      await this.#formatDisk()
    } else {
      inodes.forEach(node => this.#syncMemory(node))
    }
    this.ready = true
    return true
  }
  #syncMemory(inode) {
    this.inodeMap.set(inode.id, inode)
    if (inode.parentId) {
      if (!this.treeMap.has(inode.parentId))
        this.treeMap.set(inode.parentId, new Set())
      this.treeMap.get(inode.parentId).add(inode.id)
    }
  }
}
```

### 1.4 数据库表结构

系统使用 MySQL 8.0，共 **8 张核心表**：

#### 1.4.1 users - 用户账户表

```sql
CREATE TABLE users (
    id          INT PRIMARY KEY AUTO_INCREMENT,
    is_active   BOOLEAN DEFAULT TRUE,
    username    VARCHAR(20) DEFAULT '未设置',
    account     VARCHAR(20) UNIQUE,
    password    VARCHAR(20),
    hash_ps     VARCHAR(80),           -- bcrypt 哈希密码
    root        BOOLEAN DEFAULT FALSE,  -- 超级管理员标志
    email       VARCHAR(25) UNIQUE,
    age         INT,
    name        VARCHAR(8),
    sex         VARCHAR(1),
    phone       VARCHAR(11),
    address     VARCHAR(25),
    avatar      MEDIUMTEXT,            -- Base64 头像，最大16MB
    permission  JSON DEFAULT ('{"role":"user","first":"","second":""}')
);
```

**字段说明**：
| 字段 | 类型 | 说明 |
|------|------|------|
| id | INT PK | 自增主键 |
| account | VARCHAR(20) UNIQUE | 登录账号 |
| hash_ps | VARCHAR(80) | bcrypt 密码哈希 |
| root | BOOLEAN | 超级管理员标识 |
| email | VARCHAR(25) UNIQUE | 邮箱 |
| permission | JSON | 权限配置 `{role, first, second}` |

#### 1.4.2 user_expand - 用户扩展信息表

```sql
CREATE TABLE user_expand (
    id               INT PRIMARY KEY,
    isonline         BOOLEAN DEFAULT FALSE,
    total_active_time BIGINT,          -- 累计在线时长(秒)
    FOREIGN KEY (id) REFERENCES users(id)
);
```

#### 1.4.3 user_time_behavior - 用户时间行为表

```sql
CREATE TABLE user_time_behavior (
    user_id    INT PRIMARY KEY,
    last_login BIGINT,                 -- 最后登录时间戳
    d_mon      JSON DEFAULT ('{"logout":0,"long":0}'),  -- 周一
    d_tue      JSON DEFAULT ('{"logout":0,"long":0}'),  -- 周二
    d_wed      JSON DEFAULT ('{"logout":0,"long":0}'),  -- 周三
    d_thu      JSON DEFAULT ('{"logout":0,"long":0}'),  -- 周四
    d_fri      JSON DEFAULT ('{"logout":0,"long":0}'),  -- 周五
    d_sat      JSON DEFAULT ('{"logout":0,"long":0}'),  -- 周六
    d_sun      JSON DEFAULT ('{"logout":0,"long":0}'),  -- 周日
    time_parts JSON DEFAULT ('{}'),    -- 时段活跃度统计
    FOREIGN KEY (user_id) REFERENCES users(id)
);
```

#### 1.4.4 user_time_week - 周度时长汇总表

```sql
CREATE TABLE user_time_week (
    id            INT PRIMARY KEY AUTO_INCREMENT,
    user_id       INT,
    d_mon ~ d_sun INT DEFAULT 0,       -- 每日总时长(秒)
    week_start_ts BIGINT,              -- 周一开始时间戳
    FOREIGN KEY (user_id) REFERENCES users(id)
);
```

#### 1.4.5 sukinos_app - SukinOS 应用商店表

```sql
CREATE TABLE sukinos_app (
    resource_id   VARCHAR(120) PRIMARY KEY,
    user_id       INT NOT NULL,
    app_name      VARCHAR(20),
    created_at    DATETIME DEFAULT NOW(),
    url           VARCHAR(120) NOT NULL,
    is_private    BOOLEAN DEFAULT FALSE,
    size          INT,
    file_name     VARCHAR(50) NOT NULL,
    meta_info     JSON,
    version       VARCHAR(20) DEFAULT '0.1',
    status        VARCHAR(20) DEFAULT 'under_review',  -- active/disabled/under_review
    audit_opinion VARCHAR(255),
    FOREIGN KEY (user_id) REFERENCES users(id)
);
```

**乐观锁**：使用 SQLAlchemy `version_id_col` 机制，version 字段自动递增。

#### 1.4.6 request_logs - 请求审计日志表

```sql
CREATE TABLE request_logs (
    id               INT PRIMARY KEY AUTO_INCREMENT,

    full_url         VARCHAR(512),
    status_code      INT NOT NULL,
    success          BOOLEAN DEFAULT FALSE,
    duration_ms      INT NOT NULL,             -- 响应耗时(毫秒)

    ip               VARCHAR(64),              -- 客户端IP
    user_agent       VARCHAR(512),             -- User-Agent
    request_query    JSON,                     -- 查询参数
```
